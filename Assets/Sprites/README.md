# Sprites

Aquí están todos los assets gráficos del juego.

- `oak_woods_tileset.png` — tileset del nivel, cortado en 23 piezas
- `character/char_blue.png` — spritesheet del personaje (54 frames)
- `background/` — 3 capas de fondo (cielo, montañas, árboles)
- `decorations/` — hierba, rocas, vallas, farola, cartel
- `Items/Fruits/` — las frutas coleccionables (manzana, plátano, cereza, etc.)
- `Items/Checkpoints/` — banderas de checkpoint e inicio/fin
- `Items/Boxes/` — cajas rompibles
- `Mushroom without VFX/` — animaciones del enemigo Mushroom

Todos configurados para pixel art (Point filter, sin compresión, 16 PPU).
Para reconfigurar: `Tools > Configurar Sprites Pixel Art`.
