# Tilemaps

Aquí se guardan los Tile assets y la paleta para pintar el nivel.

- `New Palette.prefab` — la Tile Palette que uso para pintar
- `oak_woods_tileset_*.asset` — los 23 tiles individuales del tileset
- `Ground_*.asset` — tiles con nombre descriptivo (Top, Fill, Left, Right, Platform)

Para pintar el nivel: `Window > 2D > Tile Palette`, seleccionar la paleta y pintar sobre el Tilemap "Ground".
